# Kerem Kaan | Kişisel CV & Portfolyo

Kişisel CV ve portfolyo web sitemin kaynak koduna hoş geldiniz! Bu proje; yeteneklerimi, projelerimi ve profesyonel geçmişimi sergilemek için oluşturulmuş modern, etkileşimli ve kaydırılabilir bir web uygulamasıdır.

## 🌟 Genel Bakış

Web sitesi, kaydırma konumunuza göre tepki veren, dinamik ve ilgi çekici bir kullanıcı deneyimi sunan etkileyici bir **3D etkileşimli küre arka planına** sahiptir. Ön plandaki içerik, **Mobil Uygulama Geliştirme, UI/UX Tasarımı ve 3D Tasarım** konusundaki uzmanlığımı sergileyerek zarif bir şekilde ekrana gelir.

### Temel Özellikler:
- **Etkileşimli 3D Küre**: Arka planda sabit kalan ve `cobe` ile oluşturulan sürekli, performanslı bir küre.
- **Kaydırma Animasyonları**: `framer-motion` tarafından desteklenen akıcı girişler ve arka plan karartma efektleri.
- **Glassmorphism Arayüzü**: 3D kürenin hafifçe parlamasına izin verirken yeteneklerimi ve projelerimi öne çıkaran yarı saydam, şık kartlar.
- **Responsive Tasarım**: Tailwind CSS kullanılarak hem masaüstü hem de mobil görünüm için tam optimize edilmiştir.

## 💻 Yerelde Çalıştırma

Bu projeyi kendi makinenizde çalıştırmak isterseniz:

1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/Naveus/naveus-cv.git
   ```
2. Klasöre gidin:
   ```bash
   cd naveus-cv
   ```
3. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
4. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```
5. Sonucu görmek için tarayıcınızdan [http://localhost:3000](http://localhost:3000) adresini açın.
