# Kerem Kaan | Personal CV & Portfolio

Welcome to the source code of my personal CV and portfolio website! This project is a modern, interactive, and scrollable web application built to showcase my skills, projects, and professional background.

## 🌟 Overview

The website features a stunning **3D interactive globe background** that reacts to your scroll position, creating a dynamic and engaging user experience. The foreground content elegantly slides in, showcasing my expertise in **Mobile App Development, UI/UX Design, and 3D Design**.

### Key Features:
- **Interactive 3D Globe**: A continuous, performant globe rendered with `cobe` that stays fixed in the background.
- **Scroll Animations**: Smooth fade-ins and background dimming effects powered by `framer-motion`.
- **Glassmorphism UI**: Semi-transparent, sleek cards that highlight my skills and projects while allowing the 3D globe to slightly shine through.
- **Responsive Design**: Fully optimized for both desktop and mobile viewing using Tailwind CSS.

## 💻 Running Locally

If you'd like to run this project on your own machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/Naveus/naveus-cv.git
   ```
2. Navigate into the directory:
   ```bash
   cd naveus-cv
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
